# Memori

## Setup

- install dependencies `yarn`
- run dev `yarn start`

### Technologies used

- `react` (16.7.0-alpha.0) for hooks
- `final-form` & `react-final-form` as form manager
- `@reach/router` as router system
- `firebase` as login & db system
- `@lingui` as i18n provider

### Structure pattern

`scenes` / `services`
